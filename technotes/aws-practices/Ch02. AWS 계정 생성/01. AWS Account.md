## 01\. AWS 계정

### 01. AWS는 기본적으로 유료 서비스이기 때문에 계정을 생성해야 한다.
### 02. 만일 하나의 계정으로 여러 사용자가 사용해야 한다면 IAM으로 계정에 그룹 생성이 가능하며 접근 제어및 권한 관리가 가능하다.
### 03. 기본적으로 AWS 계정은 하나의 Root 사용자가 된다.
### 04. 프리티어(Free Tier) 
- 생성(가입)후 1년 동안 주요 AWS 리소스를 제한된 범위에서 무료로 사용할 수 있다.
- 기존 가입 계정 프리 티어 확인하기
